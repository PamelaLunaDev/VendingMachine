# Simple Sums

## Description
This program asks the user to type two numbers, adds them, and prints the result.

## How to run
- 1 Open the terminal in VS Code.
- 2 Compile the program:  
g++SimpleSums.cpp -o SimpleSums
- 3 Execute

.\ SimpleSums.exe

## What I learned
- Use `cin` to read numbers from the user.
- Use `cout` to print text and results.
- Understand basic arithmetic operation in C++.
- Structure of an imput/output program.
